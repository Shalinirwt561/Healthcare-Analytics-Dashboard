# Healthcare Analytics Dashboard

## Overview
Interactive Power BI dashboard built to analyze healthcare operations, revenue, patient demographics, and hospital performance.

## Tools Used
- Power BI
- Excel
- DAX

## Key KPIs
- ₹1.4B+ Net Revenue
- 40K+ Patients
- 53K+ Adjustments
- Top 5 Hospitals by Revenue

## Dashboard Features
- Revenue Trends (2019–2024)
- Patient Demographics
- Medical Conditions Analysis
- Insurance Provider Insights
- Hospital Revenue Comparison
- Age Group Segmentation

## Files Included
- Healthcare Dashboard.pbix
- Dashboard Screenshot
- Dataset (if included)

## Key Insights
- Revenue peaked during 2021–2023
- Senior patients formed largest group
- Balanced gender distribution
- Strong variation across hospitals
